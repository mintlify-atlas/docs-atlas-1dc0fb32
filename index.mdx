---
title: "JigsawStack Translation Widget"
description: "A drop-in, open source translation widget that translates any website with a single script tag. Powered by JigsawStack's contextual translation engine."
---

Add multilingual support to any website in minutes — no backend required. The JigsawStack Translation Widget detects content, translates it via JigsawStack's AI engine, and caches results locally for fast repeat visits. Works with plain HTML, React, and Next.js.

## Key features

<CardGroup cols={2}>
  <Card title="100+ languages" icon="globe">
    Translate to and from over 100 languages with a single configuration option.
  </Card>
  <Card title="Smart caching" icon="database">
    Translations are cached in localStorage so repeat visits load instantly.
  </Card>
  <Card title="Auto language detection" icon="wand-magic-sparkles">
    Automatically detect the user's browser language and translate on load.
  </Card>
  <Card title="URL-based switching" icon="link">
    Append `?lang=fr` to any URL to load the page in a specific language.
  </Card>
  <Card title="Fully customizable UI" icon="palette">
    Override CSS classes or use the built-in theme options to match your brand.
  </Card>
  <Card title="Programmatic control" icon="terminal">
    Build custom language selectors using the `window.translate()` API.
  </Card>
</CardGroup>

<Note>
  The widget requires a **public key** from your [JigsawStack dashboard](https://jigsawstack.com). Make sure translation capabilities are enabled on your key.
